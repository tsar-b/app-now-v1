# App Now Frontend

Desktop-first React shell for App Now.

This frontend lives in `frontend/` so backend generation work can continue in the
root CLI and `src/` pipeline without file conflicts.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Current Scope

- Create App flow
- Template selection
- English/Korean UI toggle
- Web/mobile target selector
- GitHub/backend-generation status surfaces

The first template source is `smart-homecare-frontend-v1` plus
`smart-homecare-backend-v1`.
